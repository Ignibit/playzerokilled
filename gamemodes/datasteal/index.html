DATA STEAL

The Data Access point is uploading sensitive data. Protect it, or try to hack it before the data upload is finished.

ATTACKERS

Locate the Data Access Point (possible positions are marked with A, B, C), and plant the Hacking Device (H) near it, by holding the trigger while holding it.

DEFENDERS

Protect the Data Access Point and eliminate the Attackers The correct DAP is marked with a red outline.
